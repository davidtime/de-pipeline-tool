The DigitalEdge Pipeline Tool is an Apache Apex-based data ingestion and processing framework. Data is read from input sources, parsed into records, mapped via a schema, optionally enriched, and persisted to a data store. The tool allows the user to create, deploy, manage and monitor systems using an intuitive graphical interface.

 

The Pipeline Tool is split into three sections.

  • The System Builder allows you to visually construct your systems by dropping in operators, customizing them, and linking them together to form a valid system.
  • The System Manager shows your systems as a glance. It provides the control interface for your system, allowing you to deploy, kill, customize, or delete your systems. Graphs and charts detailing your CPU usage as well as error logs are also available to view.
  • The Operator Library displays all the stored operators available and allows you to create your own. You can also create your own property validation that an operator can use.
