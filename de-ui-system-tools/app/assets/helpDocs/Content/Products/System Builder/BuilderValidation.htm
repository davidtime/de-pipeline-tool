Builder Validation

In order for a system to successfully run, it must meet certain criteria before it's allowed to be saved. There are a few rules a system is required to follow:

  • An Input must be followed by a Parser

  • A Binary Input must be followed by a Binary Output

  • Mappings and Enrichments are interchangeable with each other in terms of their position, but the first one used must be preceded by a Parser

  • A Mapping, Enrichment, or Parser (whichever is last in the chain) must be followed by an Output

  • Everything must be linked together
  • No circular systems
  • One input cannot accept two outputs, but one output can be split to two inputs
